Difference between Local Storage & Session Storage

Session Storage:

1: Session Storage stores data in the server as a Temporary data
2: It is used for present details on the web application
3: After reload web page the Session storage remains empty.


Local Storage:

1: Local Storage stores data in the server For further Tracking
2: It is used for present also feuture details on the web application
3: After reload web page the Local storage remains and saves Data on the Server memory.
4:Local Storage puts more Security and performance than Session storage.
